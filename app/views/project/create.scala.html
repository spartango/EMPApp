What's Next


In this project, you'll go through a series of steps to extract and process single particles from your raw micrographs. By the end, you should have a set of aligned and classified particles.

First, you'll upload a series raw micrographs. These micrographs will be the fodder for the processing steps that follow, and can be processed any number of times.

Next, you'll plan out the processing pipeline, setting the options for a series of filtering and classification steps. We'll guide you through this process to make it as painless as possible.

Finally, you'll start your processing pipeline, sending it for massively parallel execution in the cloud. After the pipeline is submitted, you'll be able to track it's progress, all the way until it's finished.
